Paste Live URL, play Live, or generate Replay from Live.
If browser playback fails, copy the URL into (MPV, VLC).
How this works
  1. You paste the Live dynamic URL: .../dynamic_playlist.m3u8?type=live
  2. Button fetches master_playlist.m3u8, finds playlist_###.m3u8, builds Replay URL with type=replay
  3. Replay URL is what you play or copy
Downloader (browser-only)
This downloads the replay stream by fetching all segments and merging them into a single file. If direct CORS fails, keep "Use CORS proxy" enabled.
Want MP3? Convert with ffmpeg (desktop): ffmpeg -i xspace_replay.ts -vn -c:a libmp3lame -b:a 128k out.mp3
Inverser (Dynamic to Replay)
Copy the result and paste into a player (MPV, VLC).
How to:
  1. Get Dynamic URL while a Space is live.
  2. Paste Dynamic URL after the Space ends.
  3. Click Get Replay URL.
  4. If the site breaks, save locally with ctrl+s and open the saved file.
Master to Dynamic + Replay
Use this in your detector workflow while a Space is live.
Paste into a player (MPV, VLC).
How to:
  1. Paste the Master URL (ends with /master_playlist.m3u8).
  2. Click Get Dynamic + Replay.
  3. You will get:
    1) Dynamic URL (for live capture workflows)
    2) Replay URL (the playlist_###.m3u8?type=replay form)
  4. If replay extraction fails, your master may be geo-blocked or the proxy is down. Dynamic still works.
  5. If the page breaks, save locally with ctrl+s and open the saved file.
